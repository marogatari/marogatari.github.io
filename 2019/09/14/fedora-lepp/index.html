The Abbreviation is FEPP

Why Fedora

New, Newer, Newest. No need to configure repos for those new software
releases, just dnf install to get satisfying newly pre-compiled binaries
along with familiar CentOS, or RedHat experiences.

Well, practically, I don’t suggest Fedora as a server
environment due to its relatively short life cycle, which is approximately
13 months. But when the time is coming, you can leap at your own risk.

GUI Administration Tool pgAdmin for PostgreSQL

PgAdmin is a powerful GUI tool for PostgreSQL. It is written in Python
and distributed as a uWSGI application. Tasks below is to run pgAdmin
with uWSGI and set Nginx as a proxy, so the admin page can be access
outside the server. That is, the pgAdmin will be configured to server mode.
